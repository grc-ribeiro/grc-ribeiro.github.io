Type specimen (pamphlet) final

  1. Please read my comments.

  2. Make the necessary adjustments

  3. follow the instructions for the submission.

You’ll submit two files.

Let’s say I have an eight-page InDesign document that looks like this.

I’d export it twice for this assignment.

Single page PDF

This is the easiest submission.

  1. File > Export

  2. Choose the proper filename and save as Adobe PDF (Print).

  3. Export all pages as single Pages, not Spreads.

  4. You don’t need crop marks or bleedings.

  5. Click on Export and you’re done with the first file.

Booklet PostScript file

Please pay attention to the following instructions.

  1. Start by making sure you have bleeding on your document. (I use 0.125in.)

  2. Then, click on File > Print Booklet….

  3. Choose 2-up Saddle Stitch and click on Print Settings….

  4. Choose PostScript® File and Adobe PDF 9.0. (There’s a great chance that you don’t have that as an option. If that’s the case, please follow the instructions from Adobe. Make sure you replace Adobe InDesign CC 2018 from step 2 in Adobe’s instruction with the version you’re using. Let me know if you need help.)

  5. On Setup, choose Paper Size: Tabloid.

  6. For orientation, you want landscape (second from left).

  7. And choose Page Position: Centered.

  8. Now you do want crop marks.

  9. And you may use the document bleed settings, if you set them up right.

  10. On the Output, choose Color: Composite CMYK,

  11. Your preview should be something like this:




  12. Click on Print.

  13. Save the .ps file as lastname-type-specimen-booklet

Evaluation

  • History

    10 points

    Are the facts context for the type family properly described and explained on the page? Did you include type designer, year, and location?

    • 10 pts Satisfactory
    • 5 pts Partially staisfactory
    • 0 pts Unsatisfactory

  • Color and visual language

    30 points

    Does the visual language appropriately reflect the references on the image collection? Do the colors effectively enhance the quality of the typeface?

    • 30 pts Satisfactory
    • 25 pts Mostly satisfactory
    • 15 pts Partially staisfactory
    • 0 pts Unsatisfactory

  • Typeface showcase

    20 points

    Can we have a good sense of the type family? Did you show the main characters? Are we able to see some nice details in its design?

    • 20 pts Satisfactory
    • 15 pts Mostly satisfactory
    • 10 pts Partially staisfactory
    • 0 pts Unsatisfactory

  • Typography

    20 points

    Are the typefaces appropriate? Does the visual hierarchy reflect the content hierarchy? Do the paragraph lines have an appropriate length? Is the type size adequate for legibility? Is the leading adequate for the type size?

    • 20 pts Satisfactory
    • 15 pts Mostly satisfactory
    • 10 pts Partially staisfactory
    • 0 pts Unsatisfactory

  • Proper files

    20 points

    Do you have files that look like mine?

    • 20 pts Satisfactory
    • 15 pts Mostly satisfactory
    • 10 pts Partially staisfactory
    • 0 pts Unsatisfactory
