Redirect file

Here’s how you can make a URL redirect to another.

The simplest way to redirect to another URL is using the HTML <meta> tag with the http-equiv parameter set to "refresh".

  1. Open a new HTML file and save it as index.html.

  2. Paste the following code into it:

    <!doctype html>  
<html>  
    <head>  
        <meta http-equiv="Cache-Control" content="no-cache, no-store, must-revalidate" />  
        <meta http-equiv="Pragma" content="no-cache" />  
        <meta http-equiv="Expires" content="0" />  
        <meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">  
        <meta http-equiv="refresh" content="0;url=https://_______.github.io/grc339/digital-magazine-proposal/">  
    </head>  
    <body>  
    </body>  
</html>

  3. Give this page a proper title. (The title of your website is the most indicated.)

  4. Replace _______ in the URL above with your teammate’s GitHub username.

  5. Place the file index.html into the folder of the original URL. For this assignment, you want save your new index.html file in your grc339/digital-magazine-proposal/ folder in the GitHub repository.

  6. Click on your name to check if it’s working.

Note

There are some disadvantages to this method and it’s discouraged to use, but for our assignment, it will work just fine. You may want to look into [other options][rd] in the future, though.
