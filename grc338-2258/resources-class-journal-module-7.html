I learned today that the tag <h1> can be used for an image.

So think about it. On our first assignment, the name of the restaurant was our h1. But in most cases, the name of the business will be a logo, not spelled out in text. So we use the logo as h1. Therefore, what should be the alt attribute for the logo? The name of the company. So you won’t need to describe the logo in the alt attribute, just make it the name ot the company.
