GRC 338: Assignments

Final project proposal

Create a fictional business and design its website. It must be a small business, with only one location. Some examples: restaurant, café, clothing shop, jewelry, pet shop, beauty salon, barber shop, music shop, record store, etc. (A design, photo, or art studio is not allowed.)

The first assignment for this final project is to write a proposal for your site.

Requirements

Introduction

Entity (company, organization, etc.)

Site description

Target audience

Who are the people who use this site?

What are their primary task and goals?

Comparative analysis

Three examples of three good websites in the same (or similar) industry.

Include two screen shots of each website.

Website content

Write the actual content for the website. Don’t use placeholders.

Make sure to break the content in web pages.
