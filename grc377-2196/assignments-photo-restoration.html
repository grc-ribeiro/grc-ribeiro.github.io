Photo restoration

Repair the image of the World War II airman, Airman.jpg.

It should be patched together and all evidence of the “rips” and scratches and missing corners and damaged edges should be cleaned up, using the pixel replacement tools and layer masks.

Replacement tools have to be used in a new layer.

Don’t erase any part of the image, use only masks.

Don’t flatten the image.

Leave all layers in the file.

Save it as PSD and submit it to PolyLearn
