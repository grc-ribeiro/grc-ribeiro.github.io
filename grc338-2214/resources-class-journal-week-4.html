It is good practice to use the <figure> tag to hold the image and the image’s caption. Unlike the alternate text, which provides a detailed description of the image, the figure’s caption can be more editorial and complement the image.

The <img> tag is an empty tag and does not have a closing tag. There are two attributes inside the <img> tag: src and alt. The src value is the URL for the image or if the file is in the same folder as the HTML document, it is the name of the file. The alt attribute is the alternate text that will be displayed if the image can’t be loaded or seen, and it is a critical aspect for improving the accessibility of your website.

Precisely.

The <src> tag provides the file name for the correct image.

Nope! There is no <src> tag. There’s a src attribute that goes inside the <img> tag.
