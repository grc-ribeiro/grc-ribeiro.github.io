The difference between margins and paddings is that margins are transparent, while paddings have the same background color as the element. Also, margins clear the area around the element, while paddings clear the area around the content and is inside of the element.

Correct.

Unlike HTML, CSS does not use tags. CSS uses declarations, and after each declaration, you need to include a semicolon. The declarations need to be within a set of curly braces. There only needs to be one CSS file for multiple HTML files if they are for the same website or use the same style.

Very important point here. There’re no tags in the CSS.
