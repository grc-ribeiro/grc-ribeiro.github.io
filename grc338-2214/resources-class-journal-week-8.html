A problem I had was getting my text to align on my website. Even though I used the same margins for my headings and paragraph, my text was unaligned as some text had bigger margins than others. From office hours, I learned that I can use rem instead of em when setting the margins. By using rem, it will ensure that all the text have the same margins because it is relative to the HTML root font size. So, even if the font size for your headings and paragraph vary, they will all have the same margin because rem uses the root font size to determine the margin. If you use em, the margin is relative to the font size of its element, which means your margins will vary depending on the font size you set your element as.

Very good explanation!

From office hours, I learned how to extend your images all the way to the edge of the screen. To do this, you would simply set the margins to 0em for the figure element in your CSS.

Correct. But when you use 0, you don’t even need unit.
