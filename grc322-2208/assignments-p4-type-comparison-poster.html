GRC 322: Assignments

Type comparison (posters)

Design three different 24″×36″ posters that inform about the differences and similarities of the type comparison webpage you worked on in week 3. Include some facts and a brief history of each typeface, but emphasize similarities and differences.

Make sure to pay attention to all the details on typography that we have covered in this class and you have learned in GRC 318, including: Typeface selection, leading, line length, kerning, tracking, grid, baseline grid, type anatomy, type hierarchy, indentation, hyphenation (just turn off), widows, orphans, proper punctuation characters, old-style figures, etc.

Since this is the exploration phase, you want the posters to be very different from each other.

Tasks

  1. Choose the best graphics to effectively show the similarities and the differences between the typefaces.

  2. Design a 24″×36″ poster, using a flexible grid, include some brief history of each typeface, but emphasize the comparison.

  3. Design another 24″×36″ poster with a very different solution for the same prompt.

  4. Design a third 24″×36″ poster with yet another very different solution for the same prompt.

Requirements

  1. History of each typeface

    For each typeface, include facts such as who design it, when, and where. Also, explain if there was a specific purpose for its design and the historical context of its creation.

  2. Comparison

    Explain, through images and diagrams, what are the biggest similarities and big differences between the two typefaces assigned to your group.

  3. Style

    For the main text use a typeface that is very different from both typefaces you are researching. Suggestion: if your typefaces are sans-serif, use a serif typeface for your copy; and vice-versa.

(Don’t include the visual references.)

Style tips

  1. Just as on the webpage, make sure your lines don’t get longer than 60 characters.

  2. Turn-off hyphenation.

  3. Check for widows.

  4. You don’t want indentation on your poster. It’s always either indentation or skipping a line. On a book, you use indentation; on a poster, you skip a line. But never both.

  5. Check your hyphens and dashes.

  6. Don’t use default fonts. You probably have access to Adobe Typekit, use it.

Evaluation

Check rubric on Canvas. Read the long description as well as each criteria.

Submission

  1. Export each of your layout twice, once showing the grid, and once without it.

  2. Make a 3‑page PDF document with all the pages exported, with the grids hidden. Name it using the name convention lastname-type-poster-grid-hidden.pdf.

  3. Make a 3‑page PDF document with all the pages exported, with the grids showing. Name it using the name convention lastname-type-poster-grid-show.pdf.

  4. Submit both files to Canvas by Monday, 26 October 2020, at 11:59pm.

This first submission will count towards your final grade for the course.
