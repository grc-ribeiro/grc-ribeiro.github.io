Absolutely.

In some textbooks I’ve seen columns separated by color. For example every other column was a block of gray from top to bottom. Based on the lecture video, is this not a good practice to follow?

It depends. It’s usually very unnecessary and just adds noise. If the table is vey long and the columns too narrow, it can be helpful. But the type alignment is usually sufficient for the columns, leaving alternating grays for the rows.

Why is it never recommended to center your content in a column? If everything is centered, won’t there be good spacing between each column?

Because when you center, there’s no alignment.
