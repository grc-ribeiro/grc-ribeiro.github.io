GRC 322: Assignments

Table design

After watching carefully the video on working with tables, make a table showing the twenty most visited cities based on arrivals, based on the data from this page.

But you want to order it by the 2019 numbers and also include one column with the comparison from 2018 to 2019. This comparison can be in percentage, or net gain/loss in absolute numbers, or previous place of the city in the list.

You also you want to highlight the European cities in the list.

Tasks

  1. Go to the CNN article, scroll down to the list, and copy the data.

  2. Paste it into Excel.

  3. Clean the data, making sure you have the columns that you need.

  4. Open InDesign and place (⌘-D or Ctrl-D) the Excel table into your file.

  5. Style your table using Table Style, Cell Styles, Paragraph Styles, and Character Styles.

Requirements

  1. Styles

    You’re required to use proper table, cell, paragraph, and character styles.

  2. Format

    Letter size, portrait.

  3. Data

    City, country (when applicable), 2019 numbers, 2018 numbers, comparison.

  4. Highlight

    Highlight the European cities in the list: London, Paris, Rome, and Istanbul (but not Antalya).

Type families

You may choose whatever typeface you want for this project, but I suggest using a sans‑serif. Helvetica is a possibility, but looks a little too generic. Arial, of course, is never acceptable. I used Aktiv Grotesk on my example, but I have some other suggestions: Acumin, Aller, Avenir, Bell Gothic, DIN, Effra, Foco, Franklin Gothic, Frutiger, Futura, Gotham, Inter, Interface, Interstate, Meta, Mr Eaves XL, Officina Sans, Optima, P22 Underground, Rotis Sans, Trade Gothic, Univers, and Vectora.

You may find it curious that Gill Sans is missing from the list above. I am not a big fan of its design, but it’s a legit choice and it’s Edward Tufte’s favorite. But before you use it in any of your designs, I want you to research the life of Eric Gill. If you decide you don’t want to use it, there are some similar (and in my opinion better) options on the list above.

Style tips

  1. Use your best judgments to round the numbers.

  2. Align numbers (and their headers) to right, everything else to left.

  3. If you have headers taking two lines, make sure you turn off hyphenation. (Just go ahead and turn off hyphenation regardless, please. There’s no way you’ll need it in a table.)

  4. Make sure you have enough room between lines and columns, but not much that they seem disconnected.

  5. You don’t want vertical lines between columns, alignment takes care of that.

  6. You may use either lines between rows or alternate colors. Or may decide you don’t need either. (You definitely don’t need both.)

  7. Don’t use default fonts. You probably have access to Adobe Typekit, use it.

  8. Above all, don’t try to be original, just try to be good.

Evaluation

Check rubric on Canvas. Read the long description as well as each criteria.

Submission

  1. Save your file as lastname-table.indd.

  2. Package your file and save the folder as a zip file using the name convention lastname-table.zip

  3. Submit your zip file to Canvas by Monday, 2 November 2020, at 11:59pm.

This is the only and final submission for this assignment.
