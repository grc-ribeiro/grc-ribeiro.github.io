GRC 338: Resources

FTP setup

  1. If you don’t have a FTP client on your computer, I recommend downloading the free Cyberduck for Mac or FileZilla for Windows. Download, install, and launch it.

  2. On your FTP client of choice, open a new connection.

  3. Type sftp://polylog1.calpoly.edu for Host.

  4. Use your Cal Poly username and password.

  5. Type 22 for Port.

  6. Click on Connect or Quickconnect.

  7. Open the public_html folder.

  8. Create a grc338 folder inside your public_html folder.

  9. Bookmark this page on client.

  10. For every new assignment, you will create a folder inside your grc338 folder. For example if the instructions of the assignment asks for your work to be published on the URL calpoly.edu/~username/grc338/sanluisobispo, you will create a sanluisobispo folder inside your grc338.

  11. Inside every new assignment folder (sanluisobispo in the example above), you will need a file named index.html. (index.html, in this case, is not an example. Your file will have to necessarily be named index.html to work. This is how the web works.)

  12. Suggestion: You may want to mimic the FTP structure in your own computer. Under your assignments folder for this class, you should have a folder for the new assignment with the exact name and files you will need to upload.
