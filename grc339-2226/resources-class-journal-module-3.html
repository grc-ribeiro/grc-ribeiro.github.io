Most do, though.

Most but not all browsers accept flexbox and grid, so make sure to have a backup for a working render. If the browser doesn't accept flexbox and grid, design the website as a stacked layout with a maximum width for the div container.

Exactly.

Are there other CSS features that aren’t supported by some browsers?

Well, any time there’s a new feature added to the CSS specs, new versions of the browsers need to support them. So a spec is new, fewer browsers will support it. Also, if a browser is older, fewer specs it will support.
