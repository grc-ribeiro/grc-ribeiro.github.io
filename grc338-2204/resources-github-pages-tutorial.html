GitHub Pages

Creating an account on GitHub

If you already have an account on GitHub, you skip to the next step. Otherwise, you must sign up for a GitHub account. It’s free.

  1. Go to github.com

  2. Choose a username and submit your email. (You don’t need to sign up with your Cal Poly email, and you don’t need to use your Cal Poly’s username either.)

  3. Choose a strong password that you have not used anywhere else.

  4. Click on the big, green button that reads Sign up for GitHub.

Setting up GitHub Pages

  1. Next to Repositories. click on the green button that reads New to create a new repository.

  2. Your new repository will be named: [yourusername].github.io, where you replace [yourusername] with your actual username. (Again, your GitHub username, not your Cal Poly’s username.)

Publishing each assignment on GitHub Pages

  1. First, go to the assignment instructions and copy the URL after the first slash (/).

  2. Now, log in GitHub and go to the repository named username.github.io.

  3. Click on Create new file.

  4. Paste the part of the URL that you copied from the assignment instructions in the first step of this short tutorial as the name of your file you’re creating.

  5. Include a new slash (/) at the end of it, so GitHub understands it’s a folder.

  6. Create the file index.html

  7. Scroll down and click on Commit new file.

  8. Click on Upload files.

  9. Drag and drop your files or click on choose your files to navigate to them on your computer.

  10. click on Commit new file again.

  11. Navigate back to the folder you were working now and copy the URL to test on a new tab or window of your browser.

  12. Don’t forget to submit your URL to Canvas.
