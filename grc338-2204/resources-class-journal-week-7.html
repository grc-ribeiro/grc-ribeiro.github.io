I am curious, what are the drawbacks to using negative margin and paddings in a design?

It’s a semantic issue. What you probably want to do is to make the previous element’s default margin zero. If you use negative margins instead, it may create a clash somewhere else. You want to keep your code clean and semantic for scalability.

I had trouble with trying to do block elements to chunk some float elements. I found on a mozilla page that there is a float option for “float: inline-start", which solved the problem.

That might be an interesting solution but don’t forget to test it in different browsers and devices to make sure it’s compatible.
