What’s the difference between pseudo-classes and regular classes and why do we need both?

They’re very different. Pseudo-classes are not classes at all. They are keywords added to a selector that specifies a special state of the selected elements. You don’t apply them in the HTML file. For us, :hover would be the main pseudo‑class that we will use. MDN Web Docs has a deeper explanation.
