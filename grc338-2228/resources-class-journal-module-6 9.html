What is line height, and why is it in a percentage?

Line height is the distance between a baseline and the previous one, same as leading in InDesign. In CSS, it can be in any unit, but the best is to write it in percentage. Usually the browser defaults to 120%, which means 1.2rem if the font-size is 1rem or 1.8rem if the font-size is 1.5rem. When you write in percentage, you can change the font‑size, and the line height will adapt automatically without having to change the value for line-height.
