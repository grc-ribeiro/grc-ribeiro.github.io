How common would you say the mobile first approach to web design is? It makes a lot of sense to do it that way, so you’d think at this point it would be pretty widespread, but I feel like even today I see so many websites that do not adapt to mobile at all.

It’s not even close to be as common as it should. It’s surprising and almost infuriating that it’s still the case more than ten years after Luke Wroblewski’s Mobile first book.

Correct.

There was an issue with accessibility on mobile devices. Not only did people want to improve that experience, but they made it a good idea to focus on that experience first and then modify it for desktop. That is a pretty cool response. Whatever first could be applied to a lot of accessibility, social, and technological issues. For a general example, you could design something to benefit an underrepresented group and then make it work for the majority after.

Here’s the best part of the accessibility: it usually benefits everybody, not only those with special needs.
