I was thinking about how it is someone inefficient to have to go in and make all the images in a photo gallery the same size. There are plenty of ways you could automate it, but I feel like they should be a way to size or make the image transparent after a certain point in the code. There’s probably some plug-in or JavaScript thing for it somewhere.

I don’t even think you’d need JavaScript for that. You can do it with HTML and CSS. overflow: hidden; is a start. I just find it easier to crop the images, especially with Actions in Photoshop. Also, have more control.

I would want to learn more about how to keep a clean CSS file, especially when you’re dealing with different pages on the same CSS file.

Essentially, you want a systematic approach to classes in HTML and CSS.
