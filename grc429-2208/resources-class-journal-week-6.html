What’s the difference between interaction framework and wireframing?

The framework is a set of quick sketches to discuss functionality and flow; details don’t matter. Wireframes is thorough representation of the interface, minus its visual language; details do matter. In the wireframe, all the necessary functions and all desired data must be present.
