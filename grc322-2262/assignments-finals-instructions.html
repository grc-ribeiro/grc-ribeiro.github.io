GRC 322: Assignments

Finals instructions

For this final exam, you will be asked to identify the typeface in 25 different examples.

The examples will be randomly assigned from a question bank, but they will be featuring one of the typefaces below. (There’ll be repetition, and it’s unlikely that all typefaces will be featured on the examples shown to you.)

Acumin
Baskerville
Bodoni
Caslon
Clarendon
Didot
DIN
Franklin Gothic
Futura
Helvetica Neue
Interstate
Mr Eaves Mod
Neue Haas Grotesk
Sutro
Trade Gothic
Univers

I have removed Helvetica Neue from the exam, because it’s not featured in Adobe Fonts, which would put those not using a Mac in disadvantage, because the font is usually not pre‑installed in non‑Apple computers.

You have two chances to get the best score. But you should get 100% on the first try because you can search for the typefaces above on Adobe Fonts and compare with the images on each question. (Also, did you know you can customize what’s the word displayed on Adobe Fonts for each typeface? That’s how I got the screenshots for this test.)

If you want to try a second chance, only the best score will count. But the questions will be different since they’re randomly chosen.

You can also work in collaboration with someone and take the test together, but again, your questions will be different. Maybe you can take the test for one person and then for another. I wouldn’t call it cheating if it helps everybody to learn more about the typefaces.
