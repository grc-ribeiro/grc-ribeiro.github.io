What to look for: CSS

Main mistake from last summer: Layout gets cut in small screens. (Usually because the value for flex-basis in the main element (line 24) is bigger than 20em.

Reminder: flex is short-hand for flex-grow, flex-shrink, and flex-basis in this order.
