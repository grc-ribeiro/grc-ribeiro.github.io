Are .svg files better for web logos orpngs?

SVG. Make sure to crop the artboard close to the logo (use margins and paddings on the CSS if necessary) and to outline the fonts.

I had to Google how to create a simple horizontal line for a page divider. How would you do this Bruno?

It depends. You can use the horizontal rule element if it’s a thematic break. Another possibility is to use the border (or border-top, or border-bottom) property to style an element, such as the semantic section or the non‑semantic div.
