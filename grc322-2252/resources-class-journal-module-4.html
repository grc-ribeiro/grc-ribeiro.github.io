I’m still confused on type hierarchy and which families are related. Like if I use a decorative typeface for the title, what typeface is most fitting for body text?

Regardless of what typeface you chose for your title, you want a legible typeface for your text. Choosing the type families that go together is a little trickier, though. You want type families that are different enough to make sense using them together. But they need to have kind of similar voice. They’re both fun, or they’re both serious, or they’re both modern, etc. That comes with time, but using references is helpful. See if the type families have been used together or if they have been used in somewhat similar visual language.
