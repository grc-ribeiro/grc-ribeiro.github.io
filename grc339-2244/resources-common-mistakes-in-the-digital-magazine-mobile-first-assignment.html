GRC 339: Resources

Common mistakes in the Digital magazine: mobile first assignment

Here’s a non-comprehensive list of mistakes I anticipate seeing. Each mistake from this list is a ten‑penalty point that you can avoid easily.

HTML

  1. Your logo must be the only h1 of your documents.

  2. Logo must always be a link to home.

  3. The navigation must be an unordered list inside the nav element.

  4. All the img tags need an (empty, for now) alt attribute.

  5. All the story headings in this assignment must be h2.

  6. The main story must be a link to the mainstory page.

  7. The other stories must be a link to #.

CSS

  1. Images must go all the way to the edge on mobile.

  2. Text should not go too close to the edge on mobile.

  3. On wider browser windows, text must align to the images.

  4. The headings for main stories, secondary stories, and tertiary stories must have different type size.

  5. All heading must be in a generic serif font.

  6. Everything else must be in a generic sans-serif font.

  7. Make sure to style the links so they’re not underlined and blue.

Don’t forget to check the [list of easy mistakes to avoid][mistakes] from GRC 338.
