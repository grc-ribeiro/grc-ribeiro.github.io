Does this project have to be an extension of an individual branding or are we creating a magazine brand? I was a little confused by it.

I don’t understand the question. It’s a digital magazine.

It always takes me so long to choose a topic when I can do whatever I want. Do you have any tips on how to be quicker?

I don’t. Sorry.

The color palette website was fun to play around with but I kept going back and fourth with what colors I wanted to go with.

Make a decision and go with it. You can always change it later.

I learned that I really like vibrant colors and the look of tropical fish and fruits.

Sounds great!

How would you get a link on your website to open a new tab?

You’d have to add the attribute target="blank" to the a tag. But don’t. Let the user decide.

I chose 3 typefaces that I think contrast a lot from each other based on vibe which I think is a good way to create personality for the design, but it hard to perfect without really designing anything yet.

It is indeed hard to choose without designing it. But now that you chose one, you have something to start with. Then you can change it if it doesn’t work.

I found lots of cool fonts from Adobe that I’d like to use later.

Keep them in your pocket. Avoid using the same type families over and over.

Are we making a logo for our website?

Yes. But keep it simple.

Will we be able to change our typefaces/colors/etc. as we move forward with this project?

Yes.
