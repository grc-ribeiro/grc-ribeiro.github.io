CSS style

Download this zip file. It contains one HTML file, one image, and one CSS file. Your job is to style the webpage to the best if your abilites. Think about some of the design principles we talked about (e.g. concistency, color, highliting, legibility, etc.) and make the page as interesting as possible. I have started the CSS with very feel lines; feel free to use them or not.

I also included a placeholder image. You need to replace it with an appropriate image. Whatever you do, though, you may not edit the HTML file at all.

Do not change the HTML file for any reason! Not one bit!!!

Furthermore, I may have overdone it with divs and classes. Don’t feel like you need to style every single class I used in my HTML file. You may choose to ignore some of them if you think it’s the most appropriate.

For this assignment, the goal is to make it look good on laptops and desktops. You don’t need to worry about mobile devices too much.

Beyond your technical skills on the HTML and CSS files, you will be evaluated on your capacity to create a solution that is visually interesting and appropriate for the content.
