What is the different between ems and rems?

em is relative to the size of the element. Let’s say you have changed the type size of h1 to 2em and kept your p as 1em (as you should). Then, if you apply a margin of 1em to each of them, their margins will be different now. But if you apply a margin of 1rem to each of them, the margin will be the same because it’ll be using the root em, which is the default font-size for the user’s browser.

You should use the flex property instead of writing flex-grow, flex-shrink, and flex-basis individually. It is more efficient and best practice.

Correct. Thanks!
