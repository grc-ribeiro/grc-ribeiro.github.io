I’m still struggling to wrap my head around the difference between section.personal-info and .personal-info when writing down the changes we want within sections.

What you write under the section.personal-info selector applies only to section elements with the personal-info class; what you write under .personal-info applies to any element with the personal-info class. In this case, it would make no difference, since the only element with the personal-info class is a section element.
