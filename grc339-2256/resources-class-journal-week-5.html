Say I want to make just some text in a <p> tag all caps. Is there any better way to do this than to create a class for just that text?

You should use classes. If it’s the whole paragraph that will be all caps, apply the class to the p tag. Most likely, though, you don’t want the whole paragraph to be all caps. In that case, wrap the word(s) you want to be all caps in a span tag and apply the class to it.

Is there a way to avoid widows in text?

You can use a non-breaking space (&nbsp;) between the last and the previous words, for example. You can also try the text-wrap: pretty; declaration: https://developer.mozilla.org/en-US/docs/Web/CSS/text-wrap. But its support is very limited for now: https://caniuse.com/mdn-css_properties_text-wrap_pretty.
