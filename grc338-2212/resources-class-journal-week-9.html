I’m also excited to compare having built a website from HTML/CSS to a website builder like Squarespace.

You may miss the simplicity of HTML/CSS sometimes.

For small businesses such as cafes or barbershops, you don’t need that level of control of coding the website so you can use website builders instead. A web designer can help customers set up their website and teach them how to do maintenance after that.

Like I said, several of my students have made some money doing exactly that.

What I’ve learned in this class helped me understand Adobe Portfolio recently because their site builder organizes page elements using containers and semantic HTML elements like headers and footers. You can customize the max-width of containers along with other adjustments that we’ve learned how to do with CSS.

That’s why it’s still very much relevant to learn to code by hand. That’s how you understand it.
