Redirect file

Here’s how you can make a URL redirect to another.

The simplest way to redirect to another URL is using the HTML <meta> tag with the http-equiv parameter set to "refresh".

  1. Open a new HTML file and save it as index.html.

  2. Paste the following code into it:

    <!doctype html>  
<html>  
    <head>  
        <title>Untitled</title>  
        <meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">  
        <meta http-equiv="refresh" content="0;url=https://calpoly.edu/">  
    </head>  
    <body>  
    </body>  
</html>

  3. Give this page a proper title. (The title of your website is the most indicated.)

  4. Replace the URL https://calpoly.edu/with your destination URL. (For this assignment, you want to replace the Cal Poly URL with your Squarespace website URL.)

  5. Make sure your Squarespace URL looks something like https://______-______-______.squarespace.com. If you include anything after the .com part of the URL, the link will be broken.

  6. Seriously, if you have /config on your destination URL, you’ll get a zero for the assignment, because I won’t be able to see it.

  7. Place the file index.html into the folder of the original URL. For this assignment, you want save your new index.html file in your grc338/website-first folder in the GitHub repository.

  8. Click on your name to check if it’s working.

  9. Ask someone else, using their own device, to click on your name to check if it’s working.

Note

There are some disadvantages to this method and it’s discouraged to use, but for our assignment, it will work just fine. You may want to look into other options in the future, though.
