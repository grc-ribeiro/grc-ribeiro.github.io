One concept I am confused on/curious about is if there is a way to do “bulk actions” in CSS. For instance, is there a way for me to make it such that all the text and images line up at the same margin left without having to manually apply the margin to each section?

As it’s often the case, you can do it in different ways:

  1. You could list several elements separated by commas and add the same declaration to them:
    header, article, article, footer {
    margin-left: 1rem;
    }

  2. Another possibility — and my favorite — is to use Eric Meyer’s CSS reset too, getting all the margins to 0, then you can add a padding to their containing element:
    div.container {
    padding-left: 1rem;
    }
