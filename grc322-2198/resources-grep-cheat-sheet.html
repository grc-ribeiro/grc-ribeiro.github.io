GREP cheat sheet

To search for a letter, you want to use \w. It’s a wild card for any uppercase letter, any lowercase letter, and the underscore (_).

  1. To search for a number, you want to use \d. It’s a wild card for any digit, from 0 to 9.

  2. For spaces, you may just use a space.

  3. If you want to search for one or more repetition, you want the +, sign. For example, \d+ will search for any sequence of consecutive numbers.

  4. If you want to search for zero or more repetition, you want the *, sign. For example, \d* will search for any sequence of consecutive numbers or no numbers at all.

  5. Some symbols—such as ., ‑, and ?—have special meanings on the GREP syntax, to search for them, then, you need to backslash them first. For example if you search for ., you will find any character at all, but if you search for \., you will find a period.

  6. When you use the square brackets ([ and ] ), you can search for any character in the bracket. For example, [bcf]at matches the words bat , cat , and fat , but not hat or rat.

  7. The search with bracket can be combined with repetition. For example, [\w\d]+ will search for any sequence of letters and numbers.

  8. To use a captured group, you need to capture the group with parenthesis and use them later with the dollar sign ($1 for the first captured group, $2 for the second, etc.)
