What’s the difference between using <main> and <section>?

main represents the main content of the page, and you should only one per page. section is for different sections of the page. You should have more than one per page. According to MDN, “sections should always have a heading, with very few exceptions”.

During this assignment, if I didn’t know what a property was going to do, it was most effective to type in an absurd value to see what happens to the style in the browser. It made it easier to pick apart what goes with what.

I like that approach too.
