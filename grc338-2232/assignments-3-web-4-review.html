Final website: review

Read the comments for the HTML/CSS and also on the design principles rubric and make sure to make all the necessary corrections.

Make sure to avoid the simple mistakes. Each mistake from the list of simple mistakes to avoid will be a 20‑point penalty. Three or more mistakes is an automatic zero, with no further comments. It’s easy to avoid them all, though.

Submission

  1. You don’t have to submit anything on Canvas.

  2. Go to your public repository on GitHub, and open a new folder named grc338/website-final. (It’s a different URL from last week.)[➤][ght]

  3. Create a new index.html file and commit. [➤][ght]

  4. Upload all four HTML files (index.html, about.html, menu.html and location.html) to your GitHub repository. [➤][ght]

  5. Upload your CSS file.

  6. Upload all your image files too.

  7. Click on your name to check if it’s working.

Check Canvas for deadlines.
