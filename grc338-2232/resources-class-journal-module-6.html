Is linking a video the same way as linking a image onto a webpage to make it appear?

Not exactly. You can use the video element, though.

To link to another internal page (another HTML file used on that particular website), you simply use the <a> tag and set the href to whatever the HTML file name is (ex: <a href="index.html"> Home </a>). To link to an external page (a webpage from an entirely different website), you simply use the full link including the https:// (ex: <a href="https://www.google.com"> Google </a>).

That’s correct!
