How to avoid floating elements messing up with the whole layout

When you have elements that float but don’t take the whole width, other elements will try to sneak in, even if they don’t have the float property applied to them.

Applying the float: none; property should fix it, but it doesn’t (sad face).

You need to give it a clear: both; property to the elements that you don’t want floating.
