How to make the content go all the way to the edge of the page

Sometimes you notice that your images or the background color don’t go all the way to the edge of the page.

There’s a simple fix:

body {  
    margin: 0;  
}

But I’ve encouraged you to use Eric Meyer’s Reset CSS tool if you don’t want to deal with default margins messing with your document’s style.
