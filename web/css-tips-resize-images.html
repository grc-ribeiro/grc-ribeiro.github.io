How to resize image using CSS

If you try to give the img element a specific width, it will mess up with all the images in your page. You don’t want that.

There are several ways to handle this issue, but this is how I handle it.

I always have my images taking 100% of its containing element:

img {  
    width: 100%;  
}

Then, I give each containing element a specific width in em or %, depending on how the layout is.

For example, I have my logo inside h1 in the HTML files:

<h1><a href="index.html"><img src="logo.png" alt="Bruno’s Restaurant’s logo."></a></h1>

Then, I style my h1 on my CSS: 

h1 {  
    width: 3em;  
    float: left;  
}

Or I have a big image on my page:

<div class="mainimage">  
    <img src="storefront.jpg" alt="Main entrance of Bruno’s Restaurant.">  
</div>

Then, I style my div on my CSS: 

div.mainimage {  
    width: 80%;  
    margin: auto;  
}

I don’t need to style my img element any further.
