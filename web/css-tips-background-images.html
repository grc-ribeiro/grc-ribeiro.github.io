Background images

Sometimes you want to set type on the top of your images.

You want to put your type on a div:

<div class="heroimage">  
    <h2>Welcome to Bruno’s Restaurant!</h2>  
</div>

You’ve noticed that I have no image on my div. On my CSS, I can make the image its background, though:

div.heroimage {  
    background-image: url("heroimage.jpg");  
    background-repeat: no-repeat;  
    background-size: cover;  
}

Where heroimage.jpg is the image you want to use. Don’t forget to include the image file on the same folder of your HTML file.

If you prefer to have your image files on a images folder, just change your CSS to:

div.heroimage {  
    background-image: url("images/heroimage.jpg");  
    background-repeat: no-repeat;  
    background-size: cover;  
}

Also, you may need to include a padding on the div.

Try this:

div.heroimage {  
    background-image: url("images/heroimage.jpg");  
    background-repeat: no-repeat;  
    background-size: cover;  
    padding: 2em 1.5em;  
}

The first value (in my example, 2em) will be applied to both top and bottom, while the second (in my example, 1.5em) will be applied to both left and right.

Furthermore, you may have notice that we don’t use the alt attribute on background images. That’s because they’re decorative only, and if they’re missing, the content wouldn’t be compromised.
