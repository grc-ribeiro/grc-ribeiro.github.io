I am still confused about why sometimes the margins don’t align despite having the same value. I was able to manually adjust the values to align closely but is there a better way?

Yes. Since em is relative to type size, it can vary depending on the font-size applied to the element. It’s best to use rem, then, which is the root em. It’s still relative to the users’ preferences, but it doesn’t vary with the type size of the element.

Setting the font size to something other than 1em will change other values set in em, such as the margin. Instead, rem can be used.

Exactly. It’s easier.
