What is the best practice for when to use padding versus margin?

You have to think semantically if you want a space inside the element (padding) or if you want space between elements (margins).

I understand the difference between padding and margins, but when mocking up something to look like an example, I never have any idea which one I should be using. I mean, it’s blank space, how can I tell which side of the border it’s on? And how can I tell whether it’s coming from the top or bottom of either element?

I usually use margin, unless it has to be a padding. And I tend to favor bottom margin unless it has to be top. Those are not hard rules at all, but can help alleviate the indecision. And don’t hesitate to break the (non‑)rules as soon as you feel like they’re hindering rather than helping.
