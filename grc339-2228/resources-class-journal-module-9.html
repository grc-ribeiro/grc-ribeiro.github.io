You can give classes to your figures.

And any HTML element.

I struggled to get the submission form to work today. After inspecting and re-inspecting the code, I realized it’s because my submission form inputs were tied to the same CSS tag that was styling the photo gallery inputs, which had display: none. It’s so important to make sure you’re styling the right thing!

Bingo!

Make sure to give elements classes or else when you add things to the CSS, it will affect the other pages too.

Not bad tip.
